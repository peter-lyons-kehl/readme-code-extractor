# negative_tests

These tests/files do _not_ need to be under `src/bin/`, since they are _not_ a crate on their own
(they are invoked by [../negative_tests_runner/](../negative_tests_runner/) instead).

However, these tests/files are under `src/bin`, so that the structure reflects
[../(positive_tests/](../positive_tests/) for easy navigation and consistency.
